# Contributing to Awesome WebMCP

Thank you for wanting to make this list even better! 🎉

## How to contribute

1. Fork this repository
2. Add new resources in the correct section of `README.md`
3. Keep descriptions short, factual, and exciting
4. Make sure links work
5. Open a Pull Request with a clear title

New sections (Podcasts, Jobs, Conferences, etc.) are very welcome!

We especially love:
- New creative demos 
- Fresh 2026 articles and videos
- Framework integrations

PRs are reviewed quickly. Let's build the best WebMCP resource together! 🚀

Questions? Open an issue.
